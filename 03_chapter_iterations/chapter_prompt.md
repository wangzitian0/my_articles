# 章节对话提示词

```
现在执行“03 章节迭代”，一次仅处理一章。
请根据以下输入生成章节草稿：
- 章节编号与标题：
- 已确认的提纲要点：
- 对应案例/数据引用：
- 用户最新反馈/需要调整的点：

输出格式：
1. 章节简述（50 字内，提醒核心矛盾）。
2. 分节写作：每节 200 字左右，包含
   - 方文山式开头场景句
   - 吴晓波式案例（主体、时间、动作、数据、结果）
   - 券商研报式数据引用（含来源）
   - 刘震云式结尾（≤ 18 字）
3. 每节后追加节级质检表：
   - 场景感 ✅/❌
   - 案例五要素 ✅/❌
   - 数据四要素 ✅/❌
   - 视觉/互动占位 ✅/❌
4. 章尾给出章级质检状态（参照 quality_checklists.md）。
5. 更新“我”字累计次数。
```
