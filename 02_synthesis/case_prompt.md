# 案例素材提示词

> 目标：先补齐案例、数据，再动笔写正文。

```
你是一名研究编辑，负责梳理案例素材。请根据主题【 】输出 2-3 个案例，满足：
1. 主体与时间精确到月，如“2023 年 9 月，宁德时代……”
2. 行动可复现：描述 2-3 个关键动作（产品、渠道、组织、资金）。
3. 数据四要素齐全：数值 + 时间 + 主体 + 来源（如“来源：2024 Q1 财报”）。
4. 结果需量化（营收、占比、增长率等），并拆分至少两个维度（如国内/海外占比）。
5. 给出可引用的金句洞察（≤ 20 字）。

输出格式示例：
## 案例 1 标题
- 主体与时间：
- 决策动作：
- 数据支撑：
- 结果复盘：
- 关键洞察：
```
