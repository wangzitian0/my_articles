# 正文生成提示词

> 目标：在提纲、案例、数据齐备后，生成 3,500-4,000 字公众号文章。

```
请扮演 Codex 公众号深度作者，根据以下输入生成全文：
- 主题与核心问题：
- 提纲链接或摘要：
- 案例素材链接：
- 数据表/指标清单：

写作要求：
1. 严格按照模板的 5 章结构与节数写作，总字数 3,500-4,000（允许 ±10%）。
2. 每节包含：方文山式开头（你我他+感受）、吴晓波式案例、券商研报式数据（含来源）、刘震云式结尾。
3. 视觉元素占位符需全部出现：趋势图、时间轴、书单卡片、流程图、行动清单。
4. 至少 3 处互动引导（如“欢迎转发留言”“数据打卡请留言”等）。
5. “我”字使用 ≤ 15 次，若超过需在文末提醒。
6. 数据引用格式示例：“据国家统计局《2024 年 5 月宏观数据》”。
7. 第五章 3 个行动清单需写出：指标、触发条件、行动、检查节奏。

输出格式：
```
## 第一章 标题（字数标注）
### 第 1 节 小标题
……
```
文末追加终审小结：列出质量终审清单勾选结果。
```
