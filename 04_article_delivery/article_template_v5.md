# 公众号深度文章通用创作模板 V5.0

## 一、核心创作原则（500 字）
1. **真实驱动**：全部信息源可查证，数据需写出数值、时间、主体、来源。
2. **深度价值**：坚持 “一厘米宽，一公里深”，完整案例优先。
3. **用户视角**：客观描述，口语化短句，“我”字 ≤ 15 次。
4. **模块设计**：全篇 5 章结构（热点→背景→案例→趋势→行动），遵循 MECE。
5. **四维融合**：每节 200 字左右，包含方文山式开头 + 吴晓波式案例 + 券商研报式数据 + 刘震云式收束。
6. **可操作性**：第五章必须提供量化指标、触发条件、检查节奏。

---

## 二、文章结构总纲（3,500-4,000 字）

| 章节 | 核心任务 | 字数 | 节数 | 每节结构 |
| --- | --- | --- | --- | --- |
| 第一章：热点切入 | 场景共鸣 + 核心问题 | 700 字 | 3 节 | 方文山开头 + 案例 + 数据 + 刘震云结尾 |
| 第二章：背景分析 | 历史脉络 + 驱动因素 | 800 字 | 3-4 节 | 同上 |
| 第三章：案例解构 | 深度案例拆解 | 800 字 | 2-3 节 | 每节 300-400 字，案例五要素 |
| 第四章：趋势展望 | 6-18 个月推演 | 700 字 | 3 节 | 同上 |
| 第五章：行动指南 | 可执行观察清单 | 600 字 | 3 节 | 同上 + 行动清单框 |

## 三、创作工作流清单

### 阶段一：准备期
- [ ] 选题验证：3 秒钩子 + 3 重痛点
- [ ] 素材备齐：3 份权威报告 + 2 个深度案例 + 2 本书籍观点
- [ ] 数据核验：所有数据具备四要素

### 阶段二：写作期
- [ ] 逐章完成：按 1-5 章顺序写作
- [ ] 元素植入：5 处视觉标注 + 3 处互动引导
- [ ] 风格校准：实时检查四维融合

### 阶段三：审查期
- [ ] 自查：按终审清单核对
- [ ] 终审：通读全文，优化流畅度

## 四、章节结构定义

### 第一章：热点切入（700 字，3 节）
- 第 1 节：个人场景开篇，小切口案例 + 冲击数据 + 矛盾结尾。
- 第 2 节：由个人扩展到群体/行业，对比数据，引出核心问题。
- 第 3 节：直面痛点，拆解数据结构，承上启下至第二章。  
  **视觉元素**：【此处插入图表：XX 趋势图】

### 第二章：背景分析（800 字，3-4 节）
- 按时间轴、驱动因素、参与者演变、国际对比展开。
- 每节含时间感开头 + 历史节点案例 + 时间序列数据 + 规律总结。  
  **视觉元素**：【此处插入时间轴：XX 发展历程】

### 第三章：案例解构（800 字，2-3 节）
- 每节 300-400 字，包含主体时间、动作、数据、原因、结果。
- 数据需多维拆解，结尾提炼普适启发。  
  **视觉元素**：【此处插入书单卡片：延伸阅读】

### 第四章：趋势展望 (700 字，3 节)
- 分别覆盖 6 / 12 / 18 个月推演。
- 开头推演句式 + 早期信号案例 + 预测性数据 + 风险/机遇结尾。  
  **视觉元素**：【此处插入流程图：未来演变路径】

### 第五章：行动指南（600 字，3 节）
- 区分投资者/创业者/普通人群。
- 每节提供适用人群、可模仿动作、数据依据、行动清单。  
  **视觉元素**：【此处插入行动清单灰底框】

行动清单模板：
```
【我的行动清单】
• 我会盯：【具体指标】
  - 股票示例：贵州茅台(600519.SH)、中证新能源(399808)
  - 政府数据示例：国家统计局月度 CPI、央行季度货币政策报告
  - 行业数据示例：中汽协月度新能源车销量
• 触发条件：当【指标】达到【数值/事件】
• 我会做：【明确动作】
• 检查节奏：每【周/月/季度】复盘一次
```

## 五、质量终审清单
- 结构层：5 章齐备，每章 2-4 节，逻辑递进。
- 内容层：无虚构场景，案例五要素齐全，数据四要素齐全，数据 ≤ 15 组，“我”字 ≤ 15 次。
- 风格层：每节具备方文山式开头、吴晓波式案例、研报数据、刘震云式结尾。
- 增强层：5 处视觉元素、3 处互动引导、第 5 章 3 个行动清单。
- 数据层：全部数据标注时间/主体/趋势/来源。

---

## 使用说明
1. 每章可独立拆用，整体字数允许 ±10% 浮动。
2. 若素材缺失，优先补齐数据与案例再写作。
3. 行动清单必须落到可查询指标。
4. 四维融合为刚性要求。
