# 质检清单总表

## 全文级（Full Article QA）
- [ ] 结构：5 章齐备，顺序为热点→背景→案例→趋势→行动。
- [ ] 字数：总字数 3,500-4,000（允许 ±10%）。
- [ ] 风格：口语化短句，“我”字 ≤ 15 次，互动引导 ≥ 3 处。
- [ ] 数据：全篇 ≤ 15 组数据，全部标注数值/时间/主体/来源。
- [ ] 视觉：趋势图、时间轴、书单卡片、流程图、行动清单各 1 处。
- [ ] 行动清单：3 个目标人群均含指标、触发条件、操作、检查节奏。
- [ ] 终审：质量终审清单勾选完毕并留档。

## 章节级（Chapter QA）
- [ ] 场景感：开篇句含人物+感受+时间感。
- [ ] 案例：至少 1 个具备主体、时间、动作、数据、结果。
- [ ] 数据：引用 ≥ 1 组时间序列或对比数据，含来源。
- [ ] 逻辑：本章回答的核心问题清晰，承上启下。
- [ ] 视觉/互动：若本章分配有视觉位或互动引导，需落地或备注“待提供”。
- [ ] 质检记录：章级 QA 表更新，若有未过项需列整改措施。

## 小节级（Section QA）
- [ ] 方文山式开头：20 字内场景句，出现“你/我/他”任一。
- [ ] 吴晓波式案例：主体、时间精确到月，动作可复现。
- [ ] 券商研报式数据：包含数值/时间/主体/来源/趋势描述。
- [ ] 刘震云式结尾：≤ 18 字，富含哲理或反思。
- [ ] 视觉/互动占位：如本节承担视觉或互动任务需注明。
- [ ] 备注：记录本节“我”字计数及是否触发上限提示。
